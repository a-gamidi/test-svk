## Как запустить проект

Проект сделан сборкой докер Laravel Sail, поэтому его очень просто запустить командой `./vendor/bin/sail up`
После чего нужно зайти в контейнер test-svk-laravel.test-1 и запустить команды:
```
composer install
php artisan migrate
```

Откройте страницу http://localhost/, если увидите стартовую страницу Laravel, значит вы всё сделали правильно

## Описание реализации

Обращение к клиенту сделано отдельным классом, который реализует интерфейс, если появится новый шлюз, то просто создаем новый класс, имплементим интерфейс, меняем реализацию в сервис провайдере и готово.

Если поймем, что новый шлюз отдает данные в другом виде, то можно между сервисом и клиентом добавить адаптер. На данный момент в этом необходимости нет.
